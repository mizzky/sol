#### createOrderLogicの処理フローメモ

```go
createOrderLogic(ctx context.Context, qtx db.Querier, userID int64) (*db.CreateOrderRow, error)
  ↓
1️⃣ GetOrCreateCartForUser(userID)
   └─ error なら return error
  ↓
2️⃣ ListCartItemsByUser(userID)
   └─ error なら return error
  ↓
3️⃣ カート空チェック（len(items) == 0）
   └─ 空なら error: "カートが空です"
  ↓
4️⃣ 合計金額計算
   └─ items の Price を合算 → Total
  ↓
5️⃣ 各 item ループ（複数商品対応）
   a. GetProductForUpdate(productID)
      └─ sql.ErrNoRows なら error: "商品が見つかりません"
      └─ その他 error なら return error
   
   b. 在庫チェック（stock >= quantity）
      └─ 不足なら error: "在庫不足です"
  ↓
6️⃣ CreateOrder(Total)
   └─ error なら return error
  ↓
7️⃣ 各 item ループで CreateOrderItem
   └─ error なら return error
  ↓
8️⃣ 各 item ループで UpdateProductStock
   └─ error なら return error
  ↓
9️⃣ ClearCartByUser(userID)
   └─ error なら return error
  ↓
🔟 注文を return
```
